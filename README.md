## How to use

1. Add the JitPack repository in your pom.xml.

   ```xml
    <repositories>
     <repository>
         <id>jitpack.io</id>
         <url>https://jitpack.io</url>
     </repository>
    </repositories>
   ```

2. Add the dependency.

   ```xml
   <dependency>
     <groupId>com.github.luisBazanDev</groupId>
     <artifactId>coffee-files</artifactId>
     <version>1.2</version>
   </dependency>
   ```

3. Finally, use the library.
